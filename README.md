# WeatherIQ
AI-powered weather dashboard with real-time forecasts, AQI, interactive charts, maps, and geolocation built using HTML, CSS, JavaScript, Open-Meteo API, Chart.js, and Leaflet.
## Features

* Real-time weather search
* Current location weather
* 7-day weather forecast
* Air Quality Index (AQI)
* Interactive temperature chart
* Search history with Local Storage
* Interactive map using Leaflet
* Responsive glassmorphism UI

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* Open-Meteo API
* Open-Meteo Air Quality API
* Chart.js
* Leaflet.js
* Local Storage
* Geolocation API
